# flowise-personal-researcher
An AI-powered personal researcher built using Flowise that can answer questions and provide books recommendations.

## Components Used
- ChatGoogleGenerativeAI (Gemini API)
- Chat Prompt Template
- LLM Chain

### How to Run
1. Install Flowise
2. Import the JSON file in this repository
3. Add your Gemini API key
4. Start chatting!

#### API Key Required
- Google Gemini API
